# Scratch Canvas — Spec

## 개요

검은 오버레이를 긁어 숨겨진 색상 레이어를 드러내는 디지털 스크래치 아트 경험.
사용자는 프리셋 색상 레이어를 고르거나, 직접 색을 칠해 나만의 레이어를 만든 뒤 긁어낸다.

## 범위

### 포함

- 캔버스 크기 선택 (픽셀 프리셋 6종)
- 색상 레이어 준비: 프리셋 6종 선택 **또는** 크레파스 16색으로 직접 칠하기
- 스크래치: 브러시로 검은 오버레이를 긁어 색상 레이어 노출 (브러시 크기 조절)
- 처음부터: 캔버스 크기 선택 단계로 초기화
- 다시 그리기: 스크래치 단계를 유지한 채 검은 오버레이만 초기화
- 완성된 작품 PNG 저장 (긁힌 영역 + 미긁힌 검은 영역 전체 포함)

### 제외

- 로그인/계정: 개인 저장·기록 불필요, 세션 내 일회성 경험
- 공유/SNS 기능: 혼자 즐기는 경험으로 충분 (idea.md 결정)
- 되돌리기(Undo): 색칠 및 긁기 단계 모두 미제공
- 커스텀 이미지 업로드로 색상 레이어 만들기: MVP 외
- AI 이미지 생성: MVP 외
- 실시간 협업: MVP 외

## 시나리오

### 1. 캔버스 크기 선택

- **Given** — 사용자가 앱에 처음 진입한 상태
- **When** — 캔버스 크기 선택 화면이 표시됨
- **Then** — 픽셀 크기 프리셋 6종이 선택 가능한 상태로 표시된다

프리셋 목록:
| 이름 | 크기 |
|---|---|
| 소형 정사각 | 600 × 600 |
| 중형 정사각 | 800 × 800 |
| 대형 정사각 | 1200 × 1200 |
| 가로형 | 1200 × 800 |
| 세로형 | 800 × 1200 |
| 와이드 | 1600 × 900 |

성공 기준:
- [ ] 앱 진입 시 6종 캔버스 크기 옵션이 모두 표시된다
- [ ] 하나를 선택하면 색상 레이어 준비 단계로 이동한다

---

### 2. 프리셋 색상 레이어 선택

- **Given** — 캔버스 크기를 선택한 상태, 색상 레이어 준비 화면
- **When** — 사용자가 "프리셋 선택"을 고르고 6종 중 하나를 선택한다
- **Then** — 선택한 프리셋 색상 레이어가 미리보기로 표시되고, "스크래치 시작" 버튼이 활성화된다

프리셋 6종 (색상 테마):
1. 무지개 (좌→우 스펙트럼 그라데이션)
2. 야경 (짙은 남색 → 보라 → 주황 수평 그라데이션)
3. 황금빛 낙조 (노랑 → 주황 → 붉은 방사형 그라데이션)
4. 새벽 (진청 → 하늘 → 연분홍 그라데이션)
5. 숲 (초록 계열 불규칙 패턴)
6. 파스텔 (분홍·민트·연노랑 파스텔 패치워크)

성공 기준:
- [ ] 6종 프리셋 미리보기가 표시된다
- [ ] 프리셋 선택 후 "스크래치 시작" 버튼이 표시된다
- [ ] "스크래치 시작" 클릭 시 스크래치 단계로 이동한다

---

### 3. 직접 색칠하기로 색상 레이어 제작

- **Given** — 색상 레이어 준비 화면에서 "직접 칠하기"를 선택한 상태
- **When** — 흰 캔버스와 크레파스 16색 팔레트가 표시된다
- **Then** — 사용자는 색을 선택하고 캔버스 위에 자유롭게 칠할 수 있다

크레파스 16색:
흰색, 검정, 빨강, 주황, 노랑, 연두, 초록, 청록, 파랑, 남색, 보라, 분홍, 갈색, 살구색, 하늘색, 회색

성공 기준:
- [ ] 흰 캔버스와 16색 팔레트가 표시된다
- [ ] 색 선택 후 캔버스 위에 드래그하면 해당 색으로 칠해진다
- [ ] 브러시 크기 슬라이더(2~30px)로 칠하기 굵기를 조절할 수 있다
- [ ] 아무것도 칠하지 않아도 "스크래치 시작" 버튼이 활성화되어 있다 (흰 캔버스도 유효한 레이어)
- [ ] "스크래치 시작" 클릭 시 스크래치 단계로 이동한다

---

### 4. 스크래치 (핵심 경험)

- **Given** — 색상 레이어(프리셋 또는 직접 칠하기)가 준비된 상태, 검은 오버레이가 캔버스 전체를 덮고 있음
- **When** — 사용자가 캔버스 위에서 마우스를 드래그하거나 터치를 이동시킨다
- **Then** — 드래그/터치 경로를 따라 검은 오버레이가 제거되고 아래 색상 레이어가 드러난다

성공 기준:
- [ ] 캔버스 전체가 검은 오버레이로 덮인 상태로 시작된다
- [ ] 마우스 드래그 경로를 따라 오버레이가 제거되어 색상 레이어가 보인다
- [ ] 터치 드래그에서도 동일하게 동작한다
- [ ] 브러시 크기 선택지 4단계(2px / 4px / 6px / 8px)가 표시되고, 선택 후 즉시 반영된다

---

### 5. 처음부터

- **Given** — 스크래치 단계에서 일부를 긁은 상태
- **When** — "처음부터" 버튼을 클릭하고 확인한다
- **Then** — 캔버스 크기 선택 화면으로 돌아가고, 이전 작업은 사라진다

성공 기준:
- [ ] "처음부터" 클릭 시 확인 메시지가 표시된다
- [ ] 확인 시 캔버스 크기 선택 단계로 이동하고 이전 작업이 초기화된다
- [ ] 취소 시 스크래치 단계를 유지한다

---

### 6. 다시 그리기 (overlay 초기화)

- **Given** — 스크래치 단계에서 일부를 긁은 상태
- **When** — "다시 그리기" 버튼을 클릭하고 확인한다
- **Then** — 캔버스의 색상 레이어와 단계는 그대로 유지된 채, 검은 오버레이만 전체에 다시 채워진다

성공 기준:
- [ ] "다시 그리기" 클릭 시 확인 메시지가 표시된다
- [ ] 확인 시 스크래치 단계를 유지한 채 오버레이가 검은색으로 복원된다
- [ ] 취소 시 현재 상태가 유지된다
- [ ] 색상 레이어와 브러시 크기 설정은 변경되지 않는다

---

### 7. 완성 작품 PNG 저장

- **Given** — 스크래치 단계에서 사용자가 작업 중이거나 완료한 상태
- **When** — "저장" 버튼을 클릭한다
- **Then** — 긁힌 영역(색상 노출)과 미긁힌 영역(검은 오버레이)을 모두 포함한 PNG 파일이 다운로드된다

성공 기준:
- [ ] "저장" 클릭 시 PNG 파일이 로컬에 다운로드된다
- [ ] 저장된 PNG에 긁힌 영역과 검은 미긁힌 영역이 모두 포함된다
- [ ] PNG 크기는 선택한 캔버스 크기와 일치한다 (예: 800×800 선택 → 800×800px PNG)

---

## 불변 규칙

- **단방향 흐름**: 색상 레이어 준비 → 스크래치는 단방향. 스크래치 단계에서 색칠 단계로 되돌아갈 수 없다. 되돌리려면 "처음부터"로 캔버스 크기 선택 단계부터 다시 시작.
- **되돌리기 없음**: 색칠과 긁기 모두 되돌리기 미제공. 실수도 작품의 일부.
